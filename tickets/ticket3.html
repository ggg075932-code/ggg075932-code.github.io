Модуль числа, два простых неравенства с модулем. Системы вложенных отрезков.

← Назад к списку билетов

Ниже представлен подробный разбор понятий модуля и систем вложенных отрезков с соответствующими доказательствами и пояснениями.

1. Модуль числа

Определение: Модулем (абсолютной величиной) действительного числа \(a\) называется само число \(a\), если \(a \ge 0\), и число \(-a\), если \(a < 0\).

Формально:

\[ |a| = \begin{cases} a, & a \ge 0 \\ -a, & a < 0 \end{cases} \]

Свойства:

  • \(|a| = \max(-a, a)\).
  • \(|-a| = |a|\).
  • Справедливы неравенства: \(-a \le |a|\) и \(a \le |a|\).

Два простых неравенства с модулем

Для любых действительных чисел \(a\) и \(b\) справедливы следующие утверждения:

Неравенство 1 (неравенство треугольника): \(|a+b| \le |a| + |b|\).

Доказательство: Основываясь на свойствах модуля, мы знаем, что \(a \le |a|\) и \(b \le |b|\). Сложив эти неравенства, получаем \(a+b \le |a| + |b|\). Аналогично, \(-a \le |a|\) и \(-b \le |b|\), откуда \(-(a+b) \le |a| + |b|\). Поскольку обе величины, \((a+b)\) и \(-(a+b)\), не превосходят \(|a|+|b|\), то и их максимум (который по определению равен \(|a+b|\)) также не превосходит эту сумму: \(\max(-(a+b), a+b) \le |a|+|b|\).

Неравенство 2: \(||a| - |b|| \le |a-b|\).

Доказательство: Представим число \(a\) в виде \(a = (a-b) + b\). Применяя неравенство треугольника, получим: \(|a| \le |a-b| + |b|\), откуда \(|a| - |b| \le |a-b|\). Аналогично, представив \(b = (b-a) + a\), получаем: \(|b| \le |b-a| + |a|\), что дает \(|b| - |a| \le |b-a|\). Так как \(|b-a| = |a-b|\), мы имеем два неравенства: \((|a|-|b|) \le |a-b|\) и \(-(|a|-|b|) \le |a-b|\). Следовательно, \(\max(|a|-|b|, -(|a|-|b|)) \le |a-b|\), что и требовалось доказать.

2. Системы вложенных отрезков

Определение: Система числовых отрезков \(\{[a_n, b_n]\}\) называется системой вложенных отрезков (СВО), если каждый последующий отрезок содержится в предыдущем: \([a_{n+1}, b_{n+1}] \subset [a_n, b_n]\) для всех \(n \in \mathbb{N}\).

Графическое понимание: Если изобразить эти отрезки на числовой прямой, то левые концы \(a_n\) образуют неубывающую последовательность (\(a_1 \le a_2 \le \dots\)), а правые концы \(b_n\) — невозрастающую (\(b_1 \ge b_2 \ge \dots\)). При этом любой левый конец \(a_k\) всегда меньше или равен любому правому концу \(b_n\).

Теорема 1 (Принцип вложенных отрезков Кантора)

Формулировка: Всякая система вложенных отрезков имеет непустое пересечение, то есть существует хотя бы одна общая точка \(c\), принадлежащая всем отрезкам системы.

Доказательство: Рассмотрим множество левых концов \(A = \{a_n\}\) и множество правых концов \(B = \{b_n\}\). Как отмечено выше, для любых \(k\) и \(n\) выполняется \(a_k \le b_n\). Согласно аксиоме непрерывности действительных чисел, если для двух множеств выполнено такое условие, то существует число \(c\), разделяющее их: \(a_n \le c \le b_n\) для всех \(n \in \mathbb{N}\). Это неравенство означает, что точка \(c\) принадлежит каждому отрезку \([a_n, b_n]\), то есть она лежит в их пересечении.

3. Стягивающиеся системы вложенных отрезков

Определение: СВО \(\{[a_n, b_n]\}\) называется стягивающейся, если длина её отрезков стремится к нулю при росте \(n\) (\(b_n - a_n \to 0\) при \(n \to \infty\)).

Формально: \(\forall \varepsilon > 0\) существует номер \(n_{\varepsilon}\), такой что \(b_{n_{\varepsilon}} - a_{n_{\varepsilon}} < \varepsilon\).

Теорема 2 (Единственность точки для стягивающейся системы)

Формулировка: Для всякой стягивающейся СВО существует единственная точка \(c\), принадлежащая всем отрезкам системы. При этом \(c = \sup\{a_n\} = \inf\{b_n\}\).

Доказательство:

Существование: По Теореме 1 хотя бы одна общая точка \(c\) существует.

Единственность: Предположим, что существуют две различные общие точки \(c\) и \(d\). Пусть для определенности \(c < d\). Поскольку обе точки принадлежат всем отрезкам \([a_n, b_n]\), расстояние между ними \(|c-d|\) не может превышать длину любого из этих отрезков: \(|c-d| \le b_n - a_n\).

Однако по условию стягиваемости длина \(b_n - a_n\) может быть сделана сколь угодно малой (меньше любого \(\varepsilon > 0\)). Если мы выберем \(\varepsilon = \frac{|c-d|}{2}\), то получим противоречие: фиксированное положительное число \(|c-d|\) должно быть меньше любого наперед заданного \(\varepsilon\), что невозможно для \(c \neq d\).

Следовательно, \(c = d\), и общая точка единственна.

\[ |a| = \max(-a, a) \] \[ |a+b| \le |a| + |b|, \quad ||a| - |b|| \le |a-b| \] \[ \bigcap_{n=1}^{\infty} [a_n, b_n] \neq \varnothing \quad \text{(Кантор)} \] \[ \text{Если } b_n - a_n \to 0, \text{ то } \bigcap_{n=1}^{\infty} [a_n, b_n] = \{c\} \]
