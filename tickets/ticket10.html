Комплексные числа. Алгебраическая и тригонометрическая формы записи, арифметические операции на поле комплексных чисел. Формула Муавра.

Комплексные числа являются расширением множества действительных чисел \(\mathbb{R}\), позволяющим решать уравнения, не имеющие корней среди вещественных чисел (например, \(x^2 + 1 = 0\)). Они образуют поле \(\mathbb{C}\), которое включает в себя все действительные числа как подмножество.

1. Определение и алгебраическая форма записи

Определение: Комплексным числом \(z\) называется упорядоченная пара действительных чисел \((a, b)\), для которых определены специфические операции сложения и умножения.

Алгебраическая форма: \(z = a + bi\), где:

  • \(a = \operatorname{Re} z\) — действительная часть.
  • \(b = \operatorname{Im} z\) — мнимая часть.
  • \(i\) — мнимая единица, определяемая фундаментальным соотношением \(i^2 = -1\) (геометрически \(i\) соответствует паре \((0,1)\)).

Сопряженное число: Для \(z = a + bi\) сопряженным является \(\overline{z} = a - bi\).

Модуль числа: \(|z| = \sqrt{a^2 + b^2}\), что отражает расстояние от начала координат до точки \((a, b)\) на плоскости.

2. Арифметические операции в алгебраической форме

Пусть \(z_1 = a_1 + b_1 i\) и \(z_2 = a_2 + b_2 i\).

Сложение и вычитание: Выполняются поэлементно.

\[ z_1 \pm z_2 = (a_1 \pm a_2) + (b_1 \pm b_2)i \]

Умножение: Выполняется по правилу раскрытия скобок с учетом \(i^2 = -1\).

\[ z_1 \cdot z_2 = (a_1 a_2 - b_1 b_2) + (a_1 b_2 + a_2 b_1)i \]

Математическое доказательство: \((a_1 + b_1 i)(a_2 + b_2 i) = a_1 a_2 + a_1 b_2 i + b_1 a_2 i + b_1 b_2 i^2 = a_1 a_2 + (a_1 b_2 + a_2 b_1)i + b_1 b_2 (-1) = (a_1 a_2 - b_1 b_2) + (a_1 b_2 + a_2 b_1)i\).

Деление: Чтобы разделить \(z_1\) на \(z_2 \neq 0\), нужно домножить числитель и знаменатель на число, сопряженное знаменателю (\(\overline{z_2}\)).

\[ \frac{z_1}{z_2} = \frac{a_1 a_2 + b_1 b_2}{a_2^2 + b_2^2} + \frac{b_1 a_2 - a_1 b_2}{a_2^2 + b_2^2} i \]

3. Геометрическое представление и тригонометрическая форма

Графическое понимание: Комплексное число \(z = a + bi\) изображается как точка \(M(a, b)\) или вектор на комплексной плоскости, где ось \(Ox\) — действительная, а \(Oy\) — мнимая.

Параметры:

  • Радиус-вектор \(r\): Длина вектора \(r = |z| = \sqrt{a^2 + b^2}\).
  • Аргумент \(\phi\): Угол между вектором и положительным направлением оси \(Ox\). Обозначается \(\operatorname{Arg} z = \arg z + 2\pi n, n \in \mathbb{Z}\), где \(\arg z \in [0, 2\pi)\) — главное значение.

Тригонометрическая форма: Используя соотношения \(a = r \cos \phi\) и \(b = r \sin \phi\), получаем:

\[ z = r(\cos \phi + i \sin \phi) \]

4. Операции в тригонометрической форме

Эта форма наиболее удобна для умножения и деления.

Умножение: Модули перемножаются, а аргументы складываются.

\[ z_1 \cdot z_2 = r_1 r_2 (\cos(\phi_1 + \phi_2) + i \sin(\phi_1 + \phi_2)) \]

Математическое доказательство: \(z_1 z_2 = r_1(\cos \phi_1 + i \sin \phi_1) \cdot r_2(\cos \phi_2 + i \sin \phi_2) = r_1 r_2 [(\cos \phi_1 \cos \phi_2 - \sin \phi_1 \sin \phi_2) + i(\sin \phi_1 \cos \phi_2 + \cos \phi_1 \sin \phi_2)]\). По тригонометрическим формулам сложения аргументов это равно \(r_1 r_2 (\cos(\phi_1 + \phi_2) + i \sin(\phi_1 + \phi_2))\).

Деление: Модули делятся, аргументы вычитаются.

\[ \frac{z_1}{z_2} = \frac{r_1}{r_2} (\cos(\phi_1 - \phi_2) + i \sin(\phi_1 - \phi_2)) \]

5. Формула Муавра

Формула Муавра служит для возведения комплексного числа в натуральную степень \(n\).

Формулировка:

\[ z^n = [r(\cos \phi + i \sin \phi)]^n = r^n (\cos(n\phi) + i \sin(n\phi)) \]

Математическое доказательство: Доказательство легко проводится методом математической индукции на основе правила умножения в тригонометрической форме.

  1. Для \(n = 1\) формула очевидна.
  2. Пусть она верна для \(n = k\): \(z^k = r^k (\cos k\phi + i \sin k\phi)\).
  3. Тогда для \(n = k + 1\): \(z^{k+1} = z^k \cdot z = r^k (\cos k\phi + i \sin k\phi) \cdot r(\cos \phi + i \sin \phi)\).
  4. По правилу умножения (модули перемножаются, аргументы складываются): \(z^{k+1} = r^{k+1} (\cos(k\phi + \phi) + i \sin(k\phi + \phi)) = r^{k+1} (\cos((k+1)\phi) + i \sin((k+1)\phi))\).
  5. Формула доказана для любого \(n \in \mathbb{N}\). Она значительно упрощает расчеты при высоких степенях.
\[ z = a + bi = r(\cos \phi + i \sin \phi) \] \[ r = \sqrt{a^2 + b^2}, \quad \phi = \arctg \frac{b}{a} \] \[ z^n = r^n (\cos n\phi + i \sin n\phi) \quad \text{(Формула Муавра)} \]
