Понятие предела последовательности является фундаментальным для всего математического анализа, так как на нем базируются определения непрерывности, производной и интеграла.

1. Определение предела последовательности

Последовательность — это упорядоченный список чисел, где каждому натуральному числу \(n\) ставится в соответствие число \(x_n\).

Строгое определение: Число \(a\) называется пределом последовательности \(\{x_n\}\), если для любого (даже сколь угодно малого) положительного числа \(\varepsilon\) существует такое натуральное число \(N_{\varepsilon}\), что для всех номеров \(n \ge N_{\varepsilon}\) выполняется неравенство \(|x_n - a| < \varepsilon\).

\[ \lim_{n \to \infty} x_n = a \Longleftrightarrow \forall \varepsilon > 0 \; \exists N_{\varepsilon} \in \mathbb{N}: \forall n \ge N_{\varepsilon} \Longrightarrow |x_n - a| < \varepsilon \]

Графическое и интуитивное понимание: Неравенство \(|x_n - a| < \varepsilon\) можно переписать как \(a - \varepsilon < x_n < a + \varepsilon\). Это означает, что все члены последовательности, начиная с некоторого номера, попадают в \(\varepsilon\)-окрестность точки \(a\) (или «\(\varepsilon\)-коридор»).

Вне этого «коридора» может находиться лишь конечное число членов последовательности. Как бы мы ни сужали этот коридор (уменьшая \(\varepsilon\)), внутри него всегда будет находиться «хвост» последовательности — бесконечно много её членов.

2. Свойства сходящихся последовательностей

Последовательность называется сходящейся, если она имеет конечный предел.

Свойство 1: Единственность предела

У сходящейся последовательности существует только один предел.

Доказательство (от противного):

  1. Предположим, что последовательность имеет два различных предела: \(a_1\) и \(a_2\) (\(a_1 \neq a_2\)).
  2. Выберем \(\varepsilon\) настолько малым, чтобы \(\varepsilon\)-окрестности этих точек не пересекались. Например, возьмем \(\varepsilon = \frac{|a_1 - a_2|}{2}\).
  3. По определению предела, почти все члены последовательности (все, кроме конечного числа) должны попасть в окрестность \(a_1\).
  4. В то же время, почти все члены должны попасть в окрестность \(a_2\).
  5. Но так как окрестности не пересекаются, один и тот же член последовательности не может находиться в обеих сразу. Мы получили противоречие.

Свойство 2: Ограниченность

Любая сходящаяся последовательность является ограниченной.

Доказательство:

  1. Пусть \(x_n \to a\). Возьмем \(\varepsilon = 1\). Тогда существует номер \(N\), такой что для всех \(n \ge N\) выполняется \(|x_n - a| < 1\), что означает \(a-1 < x_n < a+1\).
  2. Таким образом, мы ограничили «хвост» последовательности (все члены после \(N\)).
  3. Оставшиеся члены от \(x_1\) до \(x_{N-1}\) составляют конечное множество чисел. Любое конечное множество всегда ограничено своими максимальным и минимальным значениями.
  4. Объединяя эти границы, мы найдем константу \(C\), за пределы которой не выйдет ни один член последовательности.

3. Лемма о сохранении знака

Если последовательность \(\{a_n\}\) имеет предел \(a \neq 0\), то, начиная с некоторого номера, все члены последовательности имеют тот же знак, что и число \(a\).

Доказательство (для \(a > 0\)):

  1. Возьмем в качестве \(\varepsilon\) число \(\frac{a}{2}\). Так как \(a > 0\), то \(\varepsilon > 0\).
  2. По определению предела, найдется номер \(n_0\), такой что для всех \(n > n_0\) выполняется: \(|a_n - a| < \frac{a}{2} \Longrightarrow a - \frac{a}{2} < a_n < a + \frac{a}{2} \Longrightarrow a_n > \frac{a}{2}\).
  3. Поскольку \(\frac{a}{2} > 0\), то и все \(a_n > 0\) для \(n > n_0\). Лемма доказана.

4. Теорема о переходе к пределу в неравенстве

Если две последовательности \(\{x_n\}\) и \(\{y_n\}\), начиная с некоторого момента, связаны неравенством \(x_n \le y_n\), и у них существуют конечные пределы, то эти пределы связаны таким же неравенством:

\[ \lim_{n \to \infty} x_n \le \lim_{n \to \infty} y_n \]

Доказательство (от противного):

  1. Пусть \(\lim x_n = a\) и \(\lim y_n = b\). Предположим обратное: \(a > b\).
  2. Тогда разность \(a - b > 0\). Выберем \(\varepsilon = \frac{a-b}{2}\).
  3. При больших \(n\) значения \(x_n\) будут очень близки к \(a\) (в частности, \(x_n > \frac{a+b}{2}\)), а значения \(y_n\) — близки к \(b\) (в частности, \(y_n < \frac{a+b}{2}\)).
  4. Из этого следует, что \(y_n < x_n\), что противоречит условию \(x_n \le y_n\).

Важное замечание: Строгое неравенство \(x_n < y_n\) при переходе к пределу может превратиться в нестрогое. Например, для \(x_n = 0\) и \(y_n = \frac{1}{n}\) всегда \(x_n < y_n\), но их пределы равны 0.

5. Теорема о зажатой последовательности (Теорема о двух милиционерах)

Если последовательность \(\{y_n\}\) «зажата» между двумя другими последовательностями \(\{x_n\}\) и \(\{z_n\}\), имеющими одинаковый предел \(a\), то и предел \(\{y_n\}\) также равен \(a\).

Доказательство:

  1. Дано: \(x_n \le y_n \le z_n\) для всех \(n \ge n_0\), и \(\lim x_n = \lim z_n = a\).
  2. Запишем определение предела для \(\{x_n\}\) и \(\{z_n\}\) через окрестности: для любого \(\varepsilon > 0\) с некоторого номера \(n_3\) одновременно выполняются неравенства: \(a - \varepsilon < x_n < a + \varepsilon\) и \(a - \varepsilon < z_n < a + \varepsilon\).
  3. Используя условие «зажатости», получаем цепочку: \(a - \varepsilon < x_n \le y_n \le z_n < a + \varepsilon\).
  4. Отсюда следует, что \(a - \varepsilon < y_n < a + \varepsilon\), что в точности соответствует определению предела для последовательности \(\{y_n\}\).

Графическое понимание: Если две последовательности (как два милиционера) ведут третью последовательность под руки и обе идут в один пункт назначения \(a\), то и третья последовательность неизбежно окажется там же.

\[ \lim_{n \to \infty} x_n = a \Longleftrightarrow \forall \varepsilon > 0 \; \exists N: \forall n \ge N, |x_n - a| < \varepsilon \] \[ x_n \le y_n \Longrightarrow \lim x_n \le \lim y_n \] \[ x_n \le y_n \le z_n, \; \lim x_n = \lim z_n = a \Longrightarrow \lim y_n = a \]
