Тригонометрическая форма комплексного числа. Корень натуральной степени. Решение квадратных уравнений над полем комплексных чисел.

Комплексные числа являются расширением множества действительных чисел, которое позволяет решать любые алгебраические уравнения. В поле комплексных чисел \(\mathbb{C}\) каждое число может быть представлено не только в алгебраической, но и в тригонометрической форме, что значительно упрощает такие операции, как возведение в степень и извлечение корня.

1. Тригонометрическая форма комплексного числа

Каждое комплексное число \(z = a + bi\) можно интерпретировать геометрически как точку с координатами \((a, b)\) на комплексной плоскости (плоскости Аргана), где ось \(x\) соответствует действительной части (\(\operatorname{Re} z\)), а ось \(y\) — мнимой части (\(\operatorname{Im} z\)).

Графическое понимание и параметры:

  • Модуль числа (\(r\) или \(|z|\)): Это расстояние от начала координат до точки \((a, b)\). По теореме Пифагора оно вычисляется как \(|z| = \sqrt{a^2 + b^2}\).
  • Аргумент числа (\(\phi\) или \(\operatorname{Arg} z\)): Это угол между радиус-вектором точки \(z\) и положительным направлением действительной оси. Значение аргумента из интервала \([0, 2\pi)\) (или \((-\pi, \pi]\)) называется главным значением аргумента (\(\arg z\)).

Переход к тригонометрической форме: Используя соотношения в прямоугольном треугольнике, координаты можно выразить через модуль и аргумент: \(a = r \cos \phi\) и \(b = r \sin \phi\). Подставляя эти значения в алгебраическую форму \(z = a + bi\), получаем тригонометрическую форму записи:

\[ z = r(\cos \phi + i \sin \phi) \]

Эта форма особенно удобна для умножения и деления: при умножении комплексных чисел их модули перемножаются, а аргументы складываются.

2. Корень натуральной степени из комплексного числа

Определение: Число \(w\) называется корнем \(n\)-й степени из комплексного числа \(z\), если выполняется равенство \(w^n = z\).

Математическое доказательство и формула: Согласно основной теореме алгебры, у уравнения \(w^n = z_0\) существует ровно \(n\) комплексных корней. Для их нахождения число \(z_0\) представляется в тригонометрической форме. С учетом периодичности функций \(\sin\) и \(\cos\), аргумент можно записать как \(\phi + 2\pi k\). Тогда, согласно формуле Муавра (возведение в степень), корень \(n\)-й степени извлекается следующим образом:

  • Из модуля извлекается обычный арифметический корень: \(\sqrt[n]{|z|}\).
  • Аргумент делится на \(n\), с учетом всех возможных периодов.

Общая формула корней:

\[ z_k = \sqrt[n]{|z|} \left( \cos \frac{\arg z + 2\pi k}{n} + i \sin \frac{\arg z + 2\pi k}{n} \right), \quad k = 0, 1, \dots, n-1 \]

Графическое понимание: Все \(n\) корней расположены на окружности радиуса \(R = \sqrt[n]{|z|}\) с центром в начале координат. Они являются вершинами правильного \(n\)-угольника, вписанного в эту окружность. Первый корень соответствует \(k = 0\), а каждый последующий повернут относительно предыдущего на угол \(2\pi/n\).

3. Решение квадратных уравнений над полем комплексных чисел

В поле комплексных чисел любое квадратное уравнение \(ax^2 + bx + c = 0\) (где \(a \neq 0\)) имеет решение, даже если дискриминант отрицателен.

Алгоритм решения: Для вычисления корней используется стандартная формула:

\[ x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}, \quad \text{где } D = b^2 - 4ac \]

Особенности в поле \(\mathbb{C}\):

  • Если \(D > 0\): Уравнение имеет два различных действительных корня.
  • Если \(D = 0\): Уравнение имеет один действительный корень кратности 2 (два совпадающих корня).
  • Если \(D < 0\): В действительном поле такое уравнение корней не имеет, но в комплексном поле появляются два комплексно-сопряженных корня. Поскольку \(i^2 = -1\), мы можем представить отрицательный дискриминант как \(|D| \cdot i^2\). Тогда \(\sqrt{D} = \pm i \sqrt{|D|}\).

Пример: Для уравнения \(x^2 + 2x + 2 = 0\) дискриминант \(D = 4 - 8 = -4 = (2i)^2\). Корни вычисляются как \(x = \frac{-2 \pm 2i}{2}\), что дает \(x_1 = -1 + i\) и \(x_2 = -1 - i\).

Важное замечание: В комплексных числах операция извлечения квадратного корня всегда дает два значения (например, \(\sqrt{-1} = \{i, -i\}\)), поэтому запись вида \(\sqrt{-1} = i\) является упрощенной и может привести к ошибкам в строгих вычислениях. Для многочленов с действительными коэффициентами комплексные корни всегда появляются сопряженными парами.

\[ z = r(\cos \phi + i \sin \phi) \] \[ \sqrt[n]{z} = \sqrt[n]{r} \left( \cos \frac{\phi + 2\pi k}{n} + i \sin \frac{\phi + 2\pi k}{n} \right), \quad k = 0,1,\dots,n-1 \] \[ x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}, \quad \sqrt{D} = \pm i\sqrt{|D|} \text{ при } D < 0 \]
