📘 Билет №6

Бесконечно большие и бесконечно малые последовательности, их примеры, свойства. Предел суммы, разности, произведения и частного двух числовых последовательностей.

← Назад к списку билетов

В математическом анализе поведение последовательностей при стремлении их номера к бесконечности разделяют на несколько ключевых типов. Ниже приведен подробный разбор бесконечно малых и бесконечно больших последовательностей, их свойств и арифметики пределов с соответствующими доказательствами.

1. Бесконечно малые последовательности (Б.М.П.)

Определение: Последовательность \(\{\alpha_n\}\) называется бесконечно малой, если её предел равен нулю.

Математически:

\[ \lim_{n \to \infty} \alpha_n = 0 \Longleftrightarrow \forall \varepsilon > 0 \; \exists N_{\varepsilon} \in \mathbb{N}: \forall n > N_{\varepsilon} \Longrightarrow |\alpha_n| < \varepsilon \]

Графическое понимание: Члены последовательности, начиная с некоторого номера, попадают в сколь угодно узкую трубку (коридор) шириной \(2\varepsilon\) вокруг оси \(Ox\). Вне этого коридора может находиться лишь конечное число членов.

Примеры:

  • \(\frac{1}{n}, \frac{(-1)^n}{n}\) — стандартные примеры Б.М.П.
  • \(q^n\) при \(|q| < 1\) (например, \((1/2)^n\))
  • Любая последовательность \(x_n\), сходящаяся к числу \(a\), может быть представлена как \(x_n = a + \alpha_n\), где \(\alpha_n\) — Б.М.П.

Свойства Б.М.П.

  1. Сумма/разность двух Б.М.П. есть Б.М.П.
  2. Произведение Б.М.П. на ограниченную последовательность есть Б.М.П.
  3. Произведение двух Б.М.П. есть Б.М.П.

Математическое доказательство свойства №2 (Б.М.П. ⋅ Ограниченная):

Пусть \(\lim \alpha_n = 0\), а \(\{y_n\}\) — ограничена (\(|y_n| \le C\) для всех \(n\)). Докажем, что \(\lim(\alpha_n y_n) = 0\).

  1. Зафиксируем \(\varepsilon > 0\). Нам нужно найти такое \(N\), чтобы \(|\alpha_n y_n| < \varepsilon\).
  2. Т.к. \(\alpha_n\) — Б.М.П., то для любого \(\varepsilon' > 0\) существует \(N_1\), такое что для всех \(n > N_1\) выполняется \(|\alpha_n| < \varepsilon'\).
  3. Рассмотрим произведение: \(|\alpha_n y_n| = |\alpha_n| \cdot |y_n| < \varepsilon' \cdot C\).
  4. Выберем \(\varepsilon' = \frac{\varepsilon}{C}\) (если \(C > 0\)). Тогда для всех \(n > N_1\): \(|\alpha_n y_n| < \frac{\varepsilon}{C} \cdot C = \varepsilon\).
  5. Следовательно, произведение является Б.М.П. по определению.

2. Бесконечно большие последовательности (Б.Б.П.)

Определение: Последовательность \(\{x_n\}\) называется бесконечно большой, если её члены по модулю неограниченно возрастают.

Математически:

\[ \lim_{n \to \infty} x_n = \infty \Longleftrightarrow \forall E > 0 \; \exists N: \forall n \ge N \Longrightarrow |x_n| > E \]

Разновидности:

  • Стремление к \(+\infty\): \(\forall E > 0 \; \exists N: \forall n \ge N \Longrightarrow x_n > E\)
  • Стремление к \(-\infty\): \(\forall E > 0 \; \exists N: \forall n \ge N \Longrightarrow x_n < -E\)

Графическое понимание: Какую бы высокую (или низкую) горизонтальную границу \(E\) мы ни провели, график последовательности «пробьет» её и навсегда останется выше (или ниже) этой границы.

Примеры:

  • \(x_n = n\) (стремится к \(+\infty\))
  • \(x_n = (-1)^n \cdot n\) (стремится к \(\infty\) без знака, т.к. модули растут)

Важное различие: Любая Б.Б.П. является неограниченной, но не любая неограниченная является бесконечно большой.

Пример: \(x_n = n(1 - (-1)^n)\) принимает значения \(0, 2, 0, 4, 0, 6, \dots\). Она неограниченна (есть сколь угодно большие члены), но не является Б.Б.П., так как бесконечно много её членов равны \(0\) и не выходят за границу \(E\).

Связь Б.М.П. и Б.Б.П.: Если \(\{\alpha_n\}\) — Б.М.П. и \(\alpha_n \neq 0\), то \(\{1/\alpha_n\}\) — Б.Б.П. И наоборот.

3. Арифметические свойства пределов

Если последовательности \(\{x_n\}\) и \(\{y_n\}\) имеют конечные пределы (\(\lim x_n = a, \lim y_n = b\)), то справедливы следующие теоремы:

  • Предел суммы/разности: \(\lim (x_n \pm y_n) = a \pm b\)
  • Предел произведения: \(\lim (x_n \cdot y_n) = a \cdot b\)
  • Предел частного: \(\lim \frac{x_n}{y_n} = \frac{a}{b}\) (при \(b \neq 0\))

Доказательство предела суммы

  1. Представим последовательности через их пределы и Б.М.П.: \(x_n = a + \alpha_n\) и \(y_n = b + \beta_n\), где \(\alpha_n, \beta_n\) — Б.М.П.
  2. Рассмотрим их сумму: \(x_n + y_n = (a + \alpha_n) + (b + \beta_n) = (a + b) + (\alpha_n + \beta_n)\).
  3. По свойству Б.М.П., сумма \((\alpha_n + \beta_n)\) также является Б.М.П.
  4. Следовательно, предел суммы равен \(a + b\).

Доказательство предела произведения

  1. \(x_n y_n = (a + \alpha_n)(b + \beta_n) = ab + a\beta_n + b\alpha_n + \alpha_n \beta_n\).
  2. Разберем слагаемые:
    • \(ab\) — константа.
    • \(a\beta_n\) — Б.М.П. (константа ⋅ Б.М.П.)
    • \(b\alpha_n\) — Б.М.П.
    • \(\alpha_n \beta_n\) — Б.М.П. (произведение двух Б.М.П.)
  3. Сумма трех Б.М.П. (\(a\beta_n + b\alpha_n + \alpha_n \beta_n\)) является Б.М.П.
  4. Значит, \(\lim (x_n y_n) = ab\).

Доказательство предела частного

Для этого достаточно доказать, что \(\lim \frac{1}{y_n} = \frac{1}{b}\) при \(b \neq 0\).

  1. Рассмотрим разность: \(\frac{1}{y_n} - \frac{1}{b} = \frac{b - y_n}{b y_n} = \frac{-\beta_n}{b y_n} = \beta_n \cdot \left(-\frac{1}{b y_n}\right)\).
  2. Поскольку \(\lim y_n = b \neq 0\), то последовательность \(y_n\) «отделена от нуля» (по лемме о знаке), а значит, последовательность \(\left\{-\frac{1}{b y_n}\right\}\) ограничена.
  3. Произведение Б.М.П. \(\beta_n\) на ограниченную последовательность есть Б.М.П.
  4. Следовательно, \(\lim \frac{1}{y_n} = \frac{1}{b}\), и по теореме о произведении: \(\lim \frac{x_n}{y_n} = \lim \left(x_n \cdot \frac{1}{y_n}\right) = a \cdot \frac{1}{b} = \frac{a}{b}\).
\[ \lim_{n \to \infty} \alpha_n = 0 \quad \text{(Б.М.П.)} \] \[ \lim_{n \to \infty} x_n = \infty \quad \text{(Б.Б.П.)} \] \[ \lim (x_n \pm y_n) = \lim x_n \pm \lim y_n \] \[ \lim (x_n \cdot y_n) = \lim x_n \cdot \lim y_n \] \[ \lim \frac{x_n}{y_n} = \frac{\lim x_n}{\lim y_n} \quad (\lim y_n \neq 0) \]
