Понятие предела функции описывает поведение функции вблизи определенной точки, что является основой для определения непрерывности и производной. Ниже приведено детальное описание определений предела, их эквивалентности, свойств и поведения функций на бесконечности.

1. Определения предела функции

Существует два классических подхода к определению предела функции в точке \(x_0\), которые эквивалентны между собой.

Определение по Коши (на языке \(\varepsilon - \delta\))

Число \(b\) называется пределом функции \(f(x)\) в точке \(a\), если для любого сколь угодно малого положительного числа \(\varepsilon\) существует такое положительное число \(\delta\) (зависящее от \(\varepsilon\)), что для всех \(x\) из проколотой \(\delta\)-окрестности точки \(a\) значение функции \(f(x)\) попадает в \(\varepsilon\)-окрестность точки \(b\).

Формально:

\[ \lim_{x \to a} f(x) = b \Longleftrightarrow \forall \varepsilon > 0 \; \exists \delta > 0: \forall x \in \dot{U}_{\delta}(a) \Longrightarrow f(x) \in U_{\varepsilon}(b) \]

В виде неравенств: \(0 < |x - a| < \delta \Longrightarrow |f(x) - b| < \varepsilon\).

Графическое понимание: Если мы зафиксируем «коридор» шириной \(2\varepsilon\) вдоль оси \(y\) вокруг значения \(b\), то мы всегда сможем найти такой «коридор» шириной \(2\delta\) вдоль оси \(x\) вокруг точки \(a\), что график функции (кроме, возможно, самой точки \(x = a\)) будет находиться внутри этого прямоугольника.

Определение по Гейне (на языке последовательностей)

Число \(b\) называется пределом функции \(f(x)\) при \(x \to a\), если для любой последовательности \(\{x_n\}\), сходящейся к \(a\) (где все \(x_n \neq a\)), соответствующая последовательность значений функции \(\{f(x_n)\}\) сходится к \(b\).

Формально:

\[ \lim_{x \to a} f(x) = b \Longleftrightarrow \forall \{x_n\}: x_n \to a, x_n \neq a \Longrightarrow f(x_n) \to b \]

2. Эквивалентность определений (Доказательство Коши ⟹ Гейне)

Теорема: Если число \(b\) является пределом функции по Коши, то оно является пределом и по Гейне.

Доказательство:

  1. Пусть предел функции \(f(x)\) по Коши равен \(b\). Это значит, что для любого \(\varepsilon > 0\) найдется такое \(\delta > 0\), что из условия \(0 < |x - a| < \delta\) следует \(|f(x) - b| < \varepsilon\).
  2. Рассмотрим произвольную последовательность \(\{x_n\}\), которая сходится к \(a\), причем \(x_n \neq a\) для всех \(n\).
  3. По определению предела последовательности, для указанного выше числа \(\delta > 0\) найдется такой номер \(N\), что для всех \(n \ge N\) выполняется неравенство \(|x_n - a| < \delta\).
  4. Так как по условию \(x_n \neq a\), то мы имеем \(0 < |x_n - a| < \delta\) для всех \(n \ge N\).
  5. Поскольку эти значения \(x_n\) попадают в «дельта-коридор» из определения по Коши, для них автоматически выполняется условие: \(|f(x_n) - b| < \varepsilon\).
  6. Таким образом, для любого \(\varepsilon > 0\) мы нашли номер \(N\), начиная с которого все члены последовательности \(\{f(x_n)\}\) попадают в \(\varepsilon\)-окрестность числа \(b\).
  7. Это и означает, что \(\lim_{n \to \infty} f(x_n) = b\), то есть определение по Гейне выполнено.

3. Простейшие свойства предела функции

Пределы функций обладают свойствами, аналогичными свойствам пределов последовательностей. Пусть \(\lim_{x \to a} f(x) = b\):

  • Единственность: Если функция имеет предел, то он только один.
  • Локальная ограниченность: Если функция имеет конечный предел в точке \(a\), то она ограничена в некоторой окрестности этой точки.
  • Сохранение знака: Если \(\lim_{x \to a} f(x) = b \neq 0\), то в некоторой окрестности точки \(a\) значения функции имеют тот же знак, что и число \(b\).
  • Предел промежуточной функции: Если \(h(x) \le f(x) \le g(x)\) и \(\lim h(x) = \lim g(x) = b\), то и \(\lim f(x) = b\) (теорема о двух милиционерах).

4. Арифметические свойства предела

Если существуют конечные пределы \(\lim_{x \to a} f(x) = A\) и \(\lim_{x \to a} g(x) = B\), то:

  • Предел суммы/разности: \(\lim_{x \to a} (f(x) \pm g(x)) = A \pm B\).
  • Предел произведения: \(\lim_{x \to a} (f(x) \cdot g(x)) = A \cdot B\).
  • Вынесение константы: \(\lim_{x \to a} (c \cdot f(x)) = c \cdot A\).
  • Предел частного: \(\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{A}{B}\) при условии \(B \neq 0\).

5. Предел на бесконечности и бесконечно большие функции

Предел на бесконечности

Функция может стремиться к числу \(c\), когда аргумент \(x\) уходит в бесконечность.

Определение: \(\lim_{x \to +\infty} f(x) = c\), если для любого \(\varepsilon > 0\) найдется такое число \(M\), что для всех \(x > M\) выполняется \(|f(x) - c| < \varepsilon\).

Пример: \(\lim_{x \to \infty} \frac{1}{x} = 0\), так как при очень больших \(x\) дробь становится сколь угодно малой.

Бесконечно большие функции (Б.Б.Ф.)

Функция \(f(x)\) называется бесконечно большой при \(x \to a\), если её предел равен бесконечности.

Определение: \(\forall E > 0 \; \exists \delta > 0: \forall x \in \dot{U}_{\delta}(a) \Longrightarrow |f(x)| > E\).

Это означает, что значения функции «пробивают» любую наперед заданную границу \(E\).

Примеры:

  • Функция \(f(x) = \frac{1}{x^n}\) является бесконечно большой при \(x \to 0\).
  • Многочлен \(P_n(x) = a_n x^n + \dots\) является бесконечно большой функцией при \(x \to \infty\).
  • Показательная функция \(a^x\) стремится к \(+\infty\) при \(x \to +\infty\), если \(a > 1\).
\[ \lim_{x \to a} f(x) = b \Longleftrightarrow \begin{cases} \forall \varepsilon > 0 \; \exists \delta > 0: 0<|x-a|<\delta \Rightarrow |f(x)-b|<\varepsilon & \text{(Коши)} \\ \forall \{x_n\} \to a, x_n \neq a \Rightarrow f(x_n) \to b & \text{(Гейне)} \end{cases} \] \[ \lim_{x \to \infty} \frac{1}{x} = 0, \quad \lim_{x \to 0} \frac{1}{x^n} = \infty \]
